# [Redbot v3](https://github.com/Cog-Creators/Red-DiscordBot) cogs 


## Install guide

1. Add repository:

    `[p]repo add plusycogs https://github.com/NoPlagiarism/PlusyCogs`

2.  then:

    `[p]cog install plusycogs AnyCogInRepo`

3. Enjoy **:>**
